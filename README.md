# Usage

### 1 . Format your CSV

CSV file format should have names seperated by commas (,).
**Example CSV:**

```
First,Last
First2,Last2
First3,Last3
```
### 2. Run jenny.py
```
python3 jenny.py <CSV_FILE> options
```
